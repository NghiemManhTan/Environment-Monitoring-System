✦ How it works

Simple & effective

Just 3 steps to start monitoring your environment today.

Connect sensors

ESP32 + DHT11 reads environmental data and sends it to the server every 2 seconds over WiFi.

Server processes data

The FastAPI backend receives and stores the data, automatically detecting anomalies.

Dashboard displays it

The web dashboard updates in real time. View it anytime from your browser.

✦ FAQ

Frequently asked
questions

Quick answers about how the system works.

Does it need an internet connection?

Yes — the ESP32 gateway sends readings to the server over WiFi, and the dashboard is a web app you view in any browser.

How accurate are the sensor readings?

Accuracy depends on the sensors used (temperature/humidity, soil, rain, UV modules) — typically within a few percent, which is good enough for everyday monitoring.

Can I try it without real hardware connected?

Yes. When no device has reported data recently, the dashboard automatically shows realistic simulated readings so you can explore every feature.

Is this project open source?

Yes — the full source code for both the dashboard and the ESP32 firmware is available on GitHub.
